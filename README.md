Python Basics

A collection of beginner-friendly Python programs created while learning the fundamentals of Python programming.


Topics Covered

- Variables and Data Types
- Input and Output
- Conditional Statements
- Functions
- Mathematical Operations
- User Interaction
- Basic Problem Solving

Purpose

This repository contains practice programs developed while learning Python fundamentals. The goal is to build a strong foundation in programming concepts and problem-solving.

Technologies Used

- Python 3

How to Run

1. Clone the repository:
   git clone https://github.com/AnubhavSinghZ/Python_Basics.git

2. Open the project folder.

3. Run any program:
   python filename.py

Example:
python check_grade.py
